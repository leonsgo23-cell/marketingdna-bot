# Карусели — техническая реализация

> Финальная реализация: 02.06.2026. Не менять без явного подтверждения.

## Формат скриптов (генерируется в block7_scripts.js)

Claude генерирует каждый слайд в таком формате:

```
КАДР N:
Текст поверх фото: [3-6 слов — стопкадр, хук]
Подпись к посту: [1-2 предложения]
Промпт для изображения: [EN — NO text inside image, atmospheric scene]
```

**Важно**: Используй точное название бизнеса. НИКОГДА не писать "AI-сервис".

## Парсинг скриптов (visual.js)

- `extractSlideTexts(scripts, 'carousel')` — читает "КАДР N:" → "Текст поверх фото:" → хук на изображение
- `extractSlideCaption(scripts, N)` — читает "Подпись к посту:" → Telegram caption под слайдом
- **Обратная совместимость**: старый формат "Слайд N: [длинный текст]" → первое предложение = хук, остаток = подпись

## Наложение текста на изображение (overlayTextOnImage)

- **Метод**: `text-to-svg` → SVG `<path>` контуры → `sharp` composite
- **Шрифт**: `assets/Inter-Bold.ttf` (Inter v4.0 Bold, 415KB, реальный файл в репо)
- **Поддержка языков**: RU ✅ EN ✅ LV (ā ē ī ū č ģ ķ ļ ņ š ž) ✅
- **Полоса**: `rgba(0,0,0,0.45)` — полупрозрачная (НЕ 0.72 — слишком темно)
- **Текст**: белый, Inter Bold, max 6 слов через `wordSlice()`
- **Позиция**: `position='bottom'` (по умолчанию)

## Что НЕЛЬЗЯ делать (проверено, не работает)

- `@napi-rs/canvas fillText` — fontconfig не работает в Railway nixpacks
- Sharp + SVG `<text>` — librsvg не рендерит TTF шрифты (показывает каракули)
- Загружать Inter-Bold.ttf из интернета — однажды вернуло HTML 404 вместо шрифта
